<h1 align="center">
  📖 Sobre o Projeto
</h1> 

<div align="left"> 
  <p> 
    O SOA é um sistema desenvolvido (para uma empresa parceira) com a finalidade de automatizar a elaboração de orçamentos personalizados, substituindo um processo manual por uma solução digital mais eficiente.
    A aplicação permite centralizar os dados de insumos em um único ambiente, eliminando a necessidade de consultas externas, além de realizar automaticamente os cálculos de aproveitamento e rendimento de materiais.
    Com base nessas informações, o sistema gera orçamentos de forma rápida, precisa e padronizada, auxiliando operadores no processo de precificação e melhorando a agilidade no atendimento ao cliente.
    Dessa forma, o SOA reduz o tempo de resposta, minimiza erros de cálculo, diminui a sobrecarga de trabalho e contribui para aumentar a competitividade da empresa no mercado.
  </p>
</div>
